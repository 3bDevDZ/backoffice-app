À Propos de ce Projet

Ces wireframes représentent les vues essentielles du système de gestion commerciale CommerceFlow. Ils couvrent les fonctionnalités principales du MVP (Minimum Viable Product) définies dans le cahier des charges.

Technologies Utilisées

  • HTML5 & CSS3 (Tailwind CSS)
  • Font Awesome Icons
  • Design Responsive & Mobile-Friendly
  • Interface Moderne & Intuitive
